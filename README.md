# Anime RSS 

Este es un proyecto independiente que crea listas RSS con las últimas actualizaciones de anime tomadas algunas páginas de animación.

Objetivo: facilitar la obtención de los episodios recientes en un único repositorio de feeds, indicando el estado de cada fuente (✅ activo / ❌ obsoleta).

**Fuentes y estado**

| Fuente | RSS | Estado |
|---|---:|:---:|
| JkAnime | `https://raw.githubusercontent.com/Danieruhuli/anime-rss/main/docs/jkanime-rss.xml` | ✅ Activa |
| WCO Flix | `https://raw.githubusercontent.com/Danieruhuli/anime-rss/main/docs/wco-rss.xml` | ✅ Activa |
| HiAnime | `https://raw.githubusercontent.com/Danieruhuli/anime-rss/main/docs/hianime-rss.xml` | ❌ Obsoleta (página no disponible) |
| AnimeOnegai | `https://raw.githubusercontent.com/Danieruhuli/anime-rss/main/docs/animeonegai-rss.xml` | ❌ Obsoleta (página no disponible) |

Nota: las entradas marcadas como obsoletas se conservan en el workflow como referencia histórica y para facilitar su reactivación si las fuentes vuelvan a funcionar.

Cómo usar: Copiar el link en la tabla y colocarlo en un lector de RSS a elección, yo uso The RSS Aggregator
